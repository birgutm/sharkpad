CRAWLER Module is a module that functions to fetch data such as distribution map, publications, external links and iucn endangered red list from GBIF, PubMed, GoogleScholar, BHLn and IUCN.
Installing crawler module:
1. Copy the crawler module to your /sites/all/modules/
2. Go to administrator menu -> content -> blocks.
3. Move to enable which blocks you need.
4. Configure the block to be shown only on 'Species Account' field content types (crawler requires the species name as the title of the content).
5. Save blocks.
